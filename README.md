### 安装步骤
1. 全局安装 create-react-app
   > create-react-app的相关介绍 https://zh-hans.reactjs.org/docs/create-a-new-react-app.html#create-react-app
2. 新建项目
   npx create-react-app my-app

    ```
    |   `-- robots.txt

    ```
